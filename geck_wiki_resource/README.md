### Fast workflow  in browser

1. Open: https://geckwiki.com/index.php/Special:Export

2. In **Add pages from category**, enter a category name (examples that are typically useful on GECK Wiki):

* `Console Commands`
* `Script Commands`
* `Functions`
* `Scripting`
* `GECK`
* `Fallout: New Vegas`
  (Exact names must match what GECK Wiki uses; the UI will tell you if it’s empty.)

3. Click **Add**. It will populate the page list automatically.

4. Check **Include templates** (recommended) and optionally **Include only the current revision**.

5. Click **Export** → you get an XML file.
